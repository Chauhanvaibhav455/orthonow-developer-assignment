# 📊 Analytics Events

The landing page is designed to track important user interactions using Google Tag Manager and Google Analytics 4.

| Event Name | Description |
|------------|-------------|
| page_view | Landing page loaded |
| consultation_cta_click | User clicks the primary consultation CTA |
| consultation_form_view | Consultation form becomes visible |
| consultation_form_started | User begins entering consultation details |
| consultation_form_submitted | Consultation form successfully submitted |
| consultation_success | Thank-you confirmation displayed |
| call_now_click | User clicks the Call Now button |
| faq_expand | User expands an FAQ item |

Detailed event documentation is available in:

```
task-1/gtm-schema.md
```
